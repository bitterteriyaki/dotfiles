# `bitterteriyaki/dotfiles`
Hello, this repository contains my personal dotfiles, managed with [`chezmoi`](https://www.chezmoi.io/).

## Installation
This guide assumes you have Arch Linux as your operating system.
First of all, you must have `chezmoi` [installed](https://www.chezmoi.io/install/) on your system.

After that, you have to type this command in your terminal:
```sh
$ chezmoi init bitterteriyaki --ssh
```

Your system should already have been properly configured after this command.
